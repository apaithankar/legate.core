Detailed Description

A helper class to set provenance information for the scope.

Client programs often want to attach provenance information to each of their operations and hvae it rendered in profiling outputs. In such scenarios, client programs can create a ProvenanceTracker object with the provenance string in a scope and issue operations. All the issued operations then will be associated with the provenance information, which will be attached to them in profiling outputs.
